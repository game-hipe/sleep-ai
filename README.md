В чём суть проект? В последние дни вместо кошмаров, и параличей я начал видеть сны `Наконецто!`, и вот я решил их записывать LOL

---

## Как запустить проект?
1. Настроить .env
```bash
cp env.example .env
nano .env # Вставьте ваши данные!
```

2. Docker
```bash
docker-compose build --no-cache
docker-compose up -d
```

3. Вуаля! Всё готово!.
